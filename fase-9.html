Bikin Aplikasi EXE (Desktop)

EXE itu aplikasi yang jalan di komputer, bukan di browser. Cara bikinnya lebih dari satu — pilih yang paling cocok buat kamu.

9.1

Apa itu Aplikasi EXE?

File .exe adalah format aplikasi yang jalan di Windows. Waktu kamu install Spotify, VS Code, atau game — itu semua awalnya dari file .exe. Dia jalan langsung di sistem operasi, bukan di browser.

Bedanya sama web app: web app butuh internet dan browser, EXE bisa jalan offline dan punya akses lebih dalam ke sistem komputer (file, kamera, printer, dll).

💡 Analogi: Web app itu kayak warung online — butuh internet, diakses lewat browser. EXE itu kayak toko fisik — bisa dikunjungi offline, bisa nyimpen barang sendiri (data lokal), dan punya "kunci" sendiri (installer).

9.2

Komponen Utama Aplikasi EXE

Aplikasi EXE pada dasarnya punya 3 komponen utama:

1. UI (User Interface) — tampilan yang dilihat user. Tombol, form, jendela aplikasi. Di desktop ini disebut "window" atau "GUI" (Graphical User Interface).

2. Logic/Business Logic — otak aplikasi. Ngurus kalkulasi, pemrosesan data, aturan bisnis. Ini kode yang "mikir".

3. Data Storage — penyimpanan data. Bisa ke file lokal, database lokal (SQLite), atau kirim ke server via internet.

ℹ️ Ditambah satu lagi untuk distribusi: Installer — file setup yang memudahkan user menginstall aplikasimu. Dibuat pakai tools seperti NSIS atau Inno Setup.

9.3

Cara Termudah Bikin EXE: Electron

Electron adalah framework yang bikin kamu bisa bikin aplikasi desktop pakai HTML, CSS, dan JavaScript — bahasa yang udah kamu pelajari di Fase 8!

Cara kerjanya simpel: Electron pada dasarnya "membungkus" web app kamu jadi aplikasi desktop. Hasilnya bisa di-export jadi .exe untuk Windows, .dmg untuk Mac, atau .deb untuk Linux.

Aplikasi terkenal yang dibikin pakai Electron: VS Code, Slack, Discord, WhatsApp Desktop. Serius.

💡 Alternatif lebih ringan: Tauri — mirip Electron tapi ukuran filenya jauh lebih kecil. Butuh sedikit belajar Rust, tapi hasilnya lebih efisien. Atau pakai PyInstaller kalau kamu mau bikin EXE dari Python.

9.4

Cara Bikin EXE dari Python: PyInstaller

Kalau kamu prefer Python, ada PyInstaller — tool yang bisa convert script Python (.py) jadi file .exe yang bisa dijalankan tanpa perlu install Python dulu.

Untuk UI-nya, Python punya beberapa pilihan: Tkinter (built-in, paling simpel), PyQt (lebih modern, lebih kompleks), atau CustomTkinter (tampilan modern, mudah dipelajari).

⚠️ Catatan: EXE dari Electron ukurannya besar (50–150MB) karena membawa Chromium di dalamnya. Kalau ukuran file penting, pertimbangkan Tauri atau PyInstaller yang jauh lebih kecil.

9.5

Distribusi dan Update Aplikasi EXE

Setelah .exe jadi, gimana cara user dapetinnya? Ada beberapa opsi:

Direct Download — taruh file .exe di Google Drive, GitHub Releases, atau website sendiri. User tinggal download dan install.

GitHub Releases — cara paling profesional dan gratis. Setiap versi baru, upload file .exe baru ke sini.

Sistem Auto-Update — kalau pakai Electron, ada library electron-updater yang bisa bikin aplikasimu auto-update seperti Chrome atau VS Code.

💡 Tips distribusi: Sebelum distribusi, test dulu di komputer yang bersih (belum pernah install app kamu). Sering ada dependency yang ketinggalan dan bikin error di komputer orang lain.
