5.1

Apa itu API? Analogi Sederhana

Sebelum masuk ke Claude API, kita pahami dulu apa itu API secara umum.

API itu singkatan dari Application Programming Interface. Ribet kedengarannya, tapi analoginya simpel banget: API itu kayak remote control.

Bayangin Claude itu TV canggih. Kalau kamu mau nonton, kamu perlu remote buat ngontrolnya. API itu remote-nya — alat yang kamu pakai dari "luar" buat ngontrol atau minta sesuatu dari sebuah sistem, tanpa harus masuk ke "dapur"-nya langsung.

Jadi Claude API = cara kamu nyambungin kode/aplikasimu ke otak Claude, tanpa harus buka claude.ai secara manual.

💡 Bedanya pakai claude.ai vs pakai API: Di claude.ai kamu chat manual. Pakai API, kamu bisa "otomatisasi" — aplikasimu yang ngobrol sama Claude tanpa campur tangan kamu setiap saat.

5.2

Daftar Akun & Dapat API Key

Buat pakai Claude API, kamu butuh API Key — ini semacam "password khusus" yang ngasih akses ke sistem Anthropic. Setiap request yang kamu kirim lewat API harus pake key ini.

Cara dapetin API Key:

1. Buka console.anthropic.com
2. Daftar atau login dengan akun kamu
3. Klik menu "API Keys" di sidebar
4. Klik "Create Key" → kasih nama → copy key-nya

⚠️ JANGAN pernah share API Key kamu ke siapapun — atau commit ke GitHub tanpa disembunyiin. Kalau orang lain dapat key kamu, mereka bisa pakai quota API kamu dan kamu yang kena tagihan. Simpan di file .env dan masukkan .env ke .gitignore.
⚡ Action Item
1. Buka console.anthropic.com
2. Buat API Key baru
3. Simpan key-nya di tempat yang aman (misal: file .env)
4. Jangan pernah paste key kamu di chat Claude atau commit ke GitHub

5.3

Struktur Request API — Anatomi Pesan ke Claude

Setiap kali kamu kirim request ke Claude API, ada beberapa bagian yang wajib ada. Mari kita bedah satu per satu:

model — model Claude yang mau kamu pakai. Contoh: "claude-sonnet-4-5" atau "claude-haiku-4-5"

max_tokens — batas maksimal panjang jawaban Claude. 1 token ≈ 0.75 kata. Kalau kamu set 1000, jawabannya maksimal sekitar 750 kata.

messages — isi percakapannya. Berupa array berisi "role" (user atau assistant) dan "content" (isi pesannya).

ℹ️ Untuk pemula, fokus ke tiga bagian ini dulu. Nanti kalau udah lancar, baru explore parameter lain seperti temperature, system prompt, dll.
⚡ Contoh Request API (JavaScript)
const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "API_KEY_KAMU_DI_SINI",
    "anthropic-version": "2023-06-01"
  },
  body: JSON.stringify({
    model: "claude-haiku-4-5-20251001",
    max_tokens: 1000,
    messages: [
      { role: "user", content: "Halo Claude! Apa kabar?" }
    ]
  })
});

const data = await response.json();
console.log(data.content[0].text); // Ini jawaban Claude

5.4

Bedanya Pakai claude.ai vs Pakai API Langsung

Biar makin jelas, ini perbandingan lengkap antara dua cara akses Claude:

Claude.ai (chat biasa)
✓ Gratis (ada batas pesan)
✓ Langsung bisa dipakai, gak perlu coding
✓ Ada tampilan yang enak dipakai
✗ Manual, gak bisa diotomatisasi
✗ Gak bisa diintegrasikan ke aplikasi lain

Claude API
✓ Bisa diotomatisasi dan diintegrasikan ke aplikasimu
✓ Kontrol penuh atas semua parameter
✓ Bisa dipakai untuk ribuan request otomatis
✗ Berbayar per token yang dipakai
✗ Butuh kode untuk menggunakannya

💡 Kesimpulan simpel: Buat belajar dan kerja sehari-hari → pakai claude.ai. Buat produk atau fitur yang butuh AI otomatis → pakai API.

5.5

Hitung Estimasi Biaya per Token

Pakai Claude API itu bayarnya per token. Tapi tenang — harganya jauh lebih murah dari yang kamu bayangkan kalau diperhitungkan dengan baik.

Sebagai gambaran kasar (harga bisa berubah, cek selalu di docs resmi):

Claude Haiku: paling murah, cocok buat fitur yang banyak request tapi simpel
Claude Sonnet: harga menengah, balance antara kemampuan dan biaya
Claude Opus: paling mahal, buat task yang beneran butuh kecerdasan tinggi

ℹ️ Estimasi kasar: 1000 token ≈ 750 kata. Satu percakapan normal biasanya 200-500 token. Kalau aplikasimu dipakai 100 user per hari dengan 5 request masing-masing, itu sekitar 500 request/hari. Hitung sendiri berapa tokennya dan berapa biayanya di pricing page Anthropic.
⚡ Action Item
1. Buka: anthropic.com/pricing
2. Lihat harga per token untuk model yang mau kamu pakai
3. Estimasi kebutuhan token aplikasimu per bulan
4. Tanya Claude buat bantu hitung kalau bingung:
   "Aku mau bikin fitur X yang dipakai Y user per hari.
   Bantu aku estimasi biaya Claude API per bulannya."

5.6

Coba Kirim Prompt Pertama via API dari Browser

Ini momen yang seru — pertama kalinya kamu "ngobrol" sama Claude bukan lewat chat, tapi lewat kode. Kita coba pakai cara paling simpel: fetch API langsung dari browser console.

Langkahnya:
1. Buka browser (Chrome/Firefox)
2. Buka tab kosong
3. Klik kanan → Inspect → Console
4. Paste kode di bawah (isi API Key kamu dulu)
5. Tekan Enter dan lihat Claude jawab!

⚠️ Ini hanya untuk belajar/testing. Di aplikasi produksi, JANGAN pernah taruh API Key langsung di frontend/browser — harus lewat backend server supaya key kamu aman.
⚡ Kode Pertamamu Pakai Claude API
// Paste ini di browser console (F12 → Console)
// Ganti API_KEY_KAMU dengan key yang udah kamu buat

const apiKey = "API_KEY_KAMU";

const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": apiKey,
    "anthropic-version": "2023-06-01"
  },
  body: JSON.stringify({
    model: "claude-haiku-4-5-20251001",
    max_tokens: 200,
    messages: [
      { role: "user", content: "Halo! Kamu siapa dan kamu bisa bantu apa?" }
    ]
  })
});

const data = await response.json();
console.log("Jawaban Claude:", data.content[0].text);
