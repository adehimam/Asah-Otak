Bikin Aplikasi Web Sederhana

HTML, CSS, JavaScript — trio ini yang jadi fondasi semua website dan web app di dunia.

8.1

Tiga Komponen Utama Web App

Semua website yang kamu lihat di internet — dari yang sederhana sampai sekompleks YouTube — dibangun dari 3 bahan dasar yang sama.

HTML (HyperText Markup Language) — Ini adalah struktur atau kerangka halaman. Dia yang nentuin ada teks apa, gambar apa, tombol di mana. Tanpa HTML, halaman web gak ada wujudnya.

CSS (Cascading Style Sheets) — Ini yang bikin tampilan jadi cantik. Warna, ukuran font, layout, animasi — semua urusan CSS. HTML tanpa CSS itu kayak rumah tanpa cat dan dekorasi.

JavaScript — Ini yang bikin web jadi interaktif dan "hidup". Klik tombol → sesuatu terjadi. Form dikirim → data diproses. Semua logika dan interaksi ada di sini.

💡 Analogi bangunan: HTML = struktur beton & tembok. CSS = cat, wallpaper, furnitur. JavaScript = listrik, AC, pintu otomatis. Ketiganya perlu ada biar rumahnya layak huni dan nyaman.

8.2

Struktur File Web App

Web app yang rapi punya struktur folder yang terorganisir. Untuk pemula, mulai dari yang sederhana dulu:

index.html — halaman utama. Browser secara otomatis nyari file ini pertama kali.

style.css — semua styling ditaruh di sini.

script.js — semua kode JavaScript ditaruh di sini.

folder img/ — tempat nyimpen gambar-gambar.

ℹ️ Fun fact: Website "Belajar Claude" yang kamu baca sekarang juga pakai struktur ini! index.html, style.css, main.js — persis seperti yang udah kamu upload ke GitHub.

8.3

Elemen-Elemen HTML yang Wajib Diketahui

HTML punya ratusan elemen, tapi yang kepake sehari-hari cuma sekitar 20-an. Ini yang paling penting:

Teks: <h1>–<h6> untuk judul, <p> untuk paragraf, <span> untuk teks inline.

Layout: <div> untuk kotak pembungkus, <section> untuk bagian konten, <nav> untuk navigasi.

Interaksi: <button> untuk tombol, <input> untuk form isian, <a> untuk link.

Media: <img> untuk gambar, <video> untuk video.

💡 Tips: Gak perlu hafal semua. Yang penting tau fungsinya. Kalau lupa sintaksnya, tanya Claude atau googling "html [nama elemen]" — langsung keluar.

8.4

Deploy Web App ke Netlify

Ini udah familiar buat kamu sob! Tapi buat yang baru, ini alurnya:

1. Tulis kode di VS Code → simpan semua file

2. Upload ke GitHub repo baru

3. Sambungin repo ke Netlify → klik Deploy

4. Dalam 1–2 menit, web app kamu udah live dan bisa diakses siapapun di seluruh dunia 🌏

ℹ️ Gratis: Netlify gratis untuk proyek statis (HTML/CSS/JS tanpa backend). Bandwidth 100GB/bulan — lebih dari cukup untuk project personal.

8.5

Komponen Tambahan Web App Modern

Kalau web app kamu makin kompleks, kamu akan ketemu istilah-istilah ini:

Frontend — bagian yang dilihat user (HTML/CSS/JS). Ini yang udah kamu pelajari.

Backend — dapur di balik layar. Mengurus database, login user, logika bisnis. Pakai bahasa seperti Node.js, Python, PHP.

Database — tempat nyimpen data secara permanen. Contoh: MySQL, MongoDB, Supabase.

API — jembatan antara frontend dan backend, atau antara dua aplikasi berbeda. Kamu udah kenalan sama ini di Fase 5!

💡 Saran: Kuasai frontend dulu sampai solid sebelum belajar backend. Banyak developer sukses yang fokus jadi frontend specialist aja — dan itu sudah sangat bernilai di market.
